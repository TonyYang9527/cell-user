# cell-user


接下来我们应该把CoolWeather 项目中现有的文件提交到GitHub 上面，这就很简单了，
先将所有文件添加到版本控制中，如下所示：
git add .
然后在本地执行提交操作：
git commit -m "First commit."
最后将提交的内容同步到远程版本库，也就是GitHub 上面：
git push origin master
注意，最后一步的时候GitHub 要求输入用户名和密码来进行身份校验，这里输入我们
注册时填入的用户名和密码就可以了，如图14.13 所示。
